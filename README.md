# 🇸🇦 Saudi Electricity Consumption Analysis


## 🎯 Objective
Predict electricity consumption and classify sectors using Machine Learning.

## 📊 Models Used
- **Regression**: RandomForestRegressor (R² = 0.99)
- **Classification**: RandomForestClassifier (Accuracy = 61.39%)

## 📈 Key Results
| Model | Metric | Score |
|-------|--------|-------|
| Regression | R² Score | 0.99 |
| Classification | Accuracy | 61.39% |

## 🛠 Technologies
- Python, Pandas, Scikit-learn, Matplotlib, Seaborn

## 📁 Data Source
Saudi Electricity Consumption Data (2005-2022)
